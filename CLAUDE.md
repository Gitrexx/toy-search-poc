# Project Overview

This is a toy proof-of-concept (POC) project.

## Purpose

Designing and implementing a frontend using **Gradio** for building a search tool that searches for:
- Similar datasets based on input text
- Attributes of datasets based on input text

## Stack

- **Frontend**: Gradio (Python)
- **Use case**: Semantic/similarity search over datasets and their attributes
